В StudentArchive реализованы все действия, которые может совершать пользователь в архиве, в зависимости от прав.

UserLogging содержит всю логику по регистрации и авторизации пользователей, шифрованию пароля, а также содержит методы
для добавления и считывания пользователей из xml файла.

Пакет data предназначен для хранения дел студентов и пользователей архива.

Классы Client и Server реализуют сокетное соединение. Клиент отправляет запрос на сервер, где данный запрос
обрабатывается и отправляется ответ.
