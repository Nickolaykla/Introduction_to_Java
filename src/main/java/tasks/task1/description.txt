В пакете controllers содержится логика приложения: класс AbilitiesImpl реализует все возможные действия пользователя
в библиотеке. FileWorker - класс отвечающий за чтение и запись в файлы. Logging - отвечает за авторизацию и регистрацию.

В пакете files файлы adminsEmail и usersEmail используются для записи всех сообщений, которые пришли пользователям на
e-mail после добавления / удаления книг из библиотеки, а так же сообщения администраторам с предложениями о
добавлении книг. books и users содержат список книг и список пользователей библиотеки.

Сообщение на почту пользователя добавляется в List и записывается в файл.

Класс LibraryMenu - основное меню приложения.
